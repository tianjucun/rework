2. 了解常见的文件系统标志

    a: 追加, 文件不存在则创建

    r: 打开文件用于读取, 文件不存在则抛出错误

    r+ : 可读可写, 文件不存在, 不会创建并会抛出异常, 文件存在不会清空文件

    w: 打开文件用于写入, 文件不存在则创建, 文件存在则会覆盖原有内容

    w+: 可写可读, 文件不存在, 会自动创建文件, 文件存在则会覆盖原有内容



3. 了解常见的权限标识符

    fs.constants.
                F_OK: 检测文件是否存在

                R_OK: 检查文件是否可读

                W_OK: 检查文件是否可写
