

    广度优先：
    
            1. 遍历同一个等级节点后再遍历下一个等级的节点

            2. 例如：对于以下节点, 广度优先就是先把同一等级的(a, b)遍历完然后在遍历（a.1, b.1）, 然后再遍历(a.1.1, b.1.1)

                a     b

                   a.1   b.1
                     a.1.1  b.1.1

    深度优先遍历

            1. 会尝试对一个节点进行深度遍历, 直到该节点的尽头或者查找到结果才开始同一等级的下一个节点遍历

            2. 对于上面的案例, 回先遍历(a => a.1 => a.1.1), 然后遍历(b => b.1 => b.1.1)
